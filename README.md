# OpenClaw on Render

This repository demonstrates how to deploy an OpenClaw-based AI agent on Render's free tier using Docker.

## Setup

1. Fork or clone this repository.
2. Ensure you have a [Render](https://render.com) account.
3. Create a new Web Service on Render, connecting this repository.
4. Render will automatically detect the Dockerfile and build the container.
5. The service will be available at the provided Render URL.

## Environment Variables

Add any required environment variables in Render's dashboard (e.g., API keys for OpenClaw).
